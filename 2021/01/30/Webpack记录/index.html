Webpack记录

1、简介

简单使用 webpack，只需在一个空目录下执行 npm init -y，随即安装 npm i webpack -D webpack webpack-cli即可，一定不能忘记安装 webpack-cli（这样我们才能在 package.json 的脚本使用 webpack）

  • 针对 webpack-test 项目，已基本配置了 webpack，打包后也已经输出 dist/bundle.js，此时在项目下新建一个 index.html 文件，我们可在该 html 文件新建 <script src="dist/bundle.js"></script>，怎么一看也能满足需求，但：1、这需要我们手动引入比较麻烦，2、以后我们要是有 css 文件，其他 js 文件难道还是要每次手动引入吗？这样不优雅！此时我们希望我们的 html 文件 webpack 能给自动创建这样该多方便！这时候有一个 html-webpack-plugin 插件可以帮助我们解决此问题！
