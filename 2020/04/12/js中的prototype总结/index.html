对象的继承

上面谈到了针对函数，我们使用 new Func () 的方式，把 new 出来的函数，赋值给 p1 之后它就有了 name、foo 属性。我们可以说 p1、p2 继承了函数的一些属性。其实就是 Person.prototype.name = 'Frank'，Person.prototype.foo = function () {}，有了这些东西，p1、p2 才可以继承。

其实对象 Object 中的 prototype 属性下也有很多的方法，为啥能用这些方法，就是因为我们在通过对象字面量声明变量时继承了Object.prototype 中的一些方法。如我们判断一个变量到底是 Array 还是 Object，可以使用 Object.prototype.toString.call(var) 方法来判断，若变量是 Array，则打印为 [object Array]，若变量是 Object，则打印为 [object Object]。
