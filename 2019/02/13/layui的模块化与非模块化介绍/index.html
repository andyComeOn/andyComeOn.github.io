2. layui中模块加载问题

layui只use layer 打印form table 是undefined
layui只use table 打印form table 是OK的
layui只use form 打印form table 仅form是OK的

得出结论，只要use使用的有模块，layer就能使用（全局的），只use table，form也能使用；只ues form，table不能使用；table逻辑嵌入了form逻辑。
