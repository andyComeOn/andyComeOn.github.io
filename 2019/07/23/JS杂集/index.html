1、Window 的一些属性

  • 之前 Tom 大叔说过，定义变量只有一种方式：var a = *。若是不写 var，直接写：a = *，这等于说是在 Window 中定义了一个属性 a。

  • 定义一个函数 fn（声明式），在 Window 中多了一个属性 fn。

2、window.referrer

  • referrer 属性可返回载入当前文档的文档的 URL。举例：当前 A 页 http://localhost:8080/该页有 dom 元素：<a href="http://localhost:3000/" target="_blank">跳转到B</a> 点击 a 链接，进入 B 页面，控制台键入 document.referrer，打印：http://localhost:8080/，使用 window.location.href 亦如此！
